# ☑️ Todo Master

A sleek, minimalist Todo application built with **React**. This project focuses on a clean User Interface and efficient task management.

## 🚀 Features

- **Add Tasks:** Quickly add new items to your list.
- **Modern UI:** A centered, responsive card layout with soft shadows.
- **Interactive Design:** Hover effects and focus states for a better user experience.
- **Fast & Lightweight:** Built using Vite/React for near-instant load times.

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** CSS3 (Flexbox & Custom Properties)
- **Icons:** Emoji-based icons for a lightweight footprint.

## 📦 Installation

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/HarshShah24/ToDo-List.git](https://github.com/your-username/todo-master.git)
    ```

2.  **Navigate to the project folder:**

    ```bash
    cd todo-master
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```
